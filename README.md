## ✨ Features

- 🧠 ChatGPT AI Auto-Reply
- 🎵 YouTube Audio Downloader (.play2)
- 🎧 MP3/MP4 Support
- 📷 ViewOnce Image & Video Unlocker (.vv)
- 🧷 Sticker Converter (.sticker)
- ✍️ Auto Bio (Time-Based)
- 🕵 Typing + Recording Presence Rotation
- 👀 Auto Status Viewer + Emoji Liker
- 📖 Bible & Quran Integration
- 👑 Admin Tools / AntiLink / TagAll
- 🎭 Stylish Image-Based Menus
- 💬 Hinglish & Kiswahili Tone
- 🛡 AntiBot + Foreign Country Filter
