## 💡 ABOUT

**BeltahBot-MD** is a smart WhatsApp bot built using Node.js and Baileys API  
It features real-time AI conversations using **ChatGPT**, music tools, group admin controls, and more.

---

## 🔥 FEATURES

- 🤖 AI Chat with Hinglish, Swahili, IGIMO modes
- 📸 ViewOnce Remover (`.vv`)
- 🎵 Music Downloader (`.play2`, `.lyrics2`)
- 🖼 Image Menu & Captions
- 🛠 Admin Tools (`.tagall`, `.antilink`, `.welcome`)
- 👀 Auto Status Liker / Viewer
- ✍️ Auto Bio + Typing / Recording presence
