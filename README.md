## 🌟 𝐁𝐋𝐀𝐂𝐊𝐁𝐄𝐋𝐓𝐀𝐇 - MultiDevice WhatsApp Bot

> A sleek, AI-powered WhatsApp bot built by **Ishaq Ibrahim**.  
> Modern design, friendly UI, full support for group features, AI chats, status tools & more!

---

### 🚀 Features

- 💬 ChatGPT (Hinglish + Kiswahili slang)
- 🎵 Music download (play2, lyrics2)
- 🔐 Group admin tools
- 🎭 Sticker, meme & media tools
- 🧠 AI-based auto replies
- 📖 Bible & Quran search
- 🎨 Stylish image-based menu
- 🌀 Auto status, bio, and emoji reactions
- 🤖 Status view, auto-like & presence switch

---
